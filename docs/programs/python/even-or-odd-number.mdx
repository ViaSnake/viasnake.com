# 偶数/偶数を判定する関数

## 偶数かどうかを判定する

Pythonで引数に指定されたint値が偶数であるかを判定し、boolで返す関数です。

```python
def is_even(num: int):
    if num % 2 == 0:
        return True
    else:
        return False
```

# 奇数かどうかを判定する

Pythonで引数に指定されたint値が奇数であるかを判定し、boolで返す関数です。

```python
def is_odd(num: int):
    if num % 2 == 0:
        return False
    else:
        return True
```
