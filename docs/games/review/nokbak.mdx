# NOKBAK

<iframe src="https://store.steampowered.com/widget/655550/" frameborder="0" width="646" height="190"></iframe>

Steamに投稿したNOKBAKのレビューです。

スマブラとFPSをかけ合わせたようなゲームです。
プレイヤーをアリーナから吹っ飛ばせば相手のポイントが自分に入ってきて、そのポイントが一定数に達すれば勝利となります。
左下のダメージパーセントが大きくなればなるほど吹っ飛び率が高くなっていきます。

ダメージを与える武器は３つあります。

1. ダメージを洗える球体を発射する武器
2. 爆発する弾を発射する武器
3. レーザーのような大ダメージを与えられる武器

すべての武器がチャージ可能でチャージによりダメージや弾の速度が変化します。

このゲーム性はとても面白いのですが唯一の欠点として致命的にプレイヤーがいないです。身内で遊ぶのであれば楽しくプレイできるかと思います。
