# Pyxel tilemap viewer とは

Pyxel tilemap viewer は Pyxel 用のタイルマップ表示プログラムです。

## 必要要件

- [Python 3](https://www.python.org/)
- [Pyxel 1.4.3](https://github.com/kitao/pyxel)

*おそらく最新ではない Pyxel でも動作すると思いますが、動作保証はありません。*

## 使い方

1. `viewr.py` をダウンロード。
2. `viwer.py` と同じディレクトリに `tm.pyxres` を配置する。
3. `viewer.py` を実行すると、タイルマップが表示されます。

### 操作方法

- W: 上へ移動‌‌
- S: 下へ移動‌‌
- A: 左へ移動‌‌
- D: 右へ移動‌‌
- 0~7: タイルマップ変更
