# infra-core とは

Alflag で使っているインフラをコードで管理するためのプロジェクトです。

## プロジェクトの構成

以下のような構成になっています。

- **ansible**
    - Ansible の Playbook などを管理するディレクトリ
- **cloud-init**
    - Cloud-init の設定ファイルを管理するディレクトリ
- **scripts**
    - スクリプトを管理するディレクトリ
- **secrets**
    - シークレット情報を管理するディレクトリ
- **terraform**
    - Terraform の設定ファイルを管理するディレクトリ
