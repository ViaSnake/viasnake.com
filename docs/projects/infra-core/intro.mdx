# infra-core とは

```
$ tree -L 2 -d

│   ├── ansible-common

│   ├── group_vars
│   └── secrets
├── terraform
│   ├── secrets
│   ├── terraform-cloudflare-access

│   ├── terraform-dns
│   ├── terraform-oraclecloud
│   └── terraform-proxmox
└── tools
    ├── cloud-init
    ├── docker
    └── scripts
```
