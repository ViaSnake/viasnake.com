# kitsunebi (狐火) とは

[MCPlayNetwork](https://www.mcplay.biz/) で使われているインフラや設定などを管理するためのプロジェクトです。
例えば、サーバーの設定ファイルやプラグインの設定ファイルなどを一元管理し、サーバーの再構築や新規サーバーの構築を簡単に行えるようにすることを目的としています。

## 目指していること

kitsunebi は以下のようなことを目指しています。

* Docker コンテナの活用
* 安定性の向上
* ポータビリティの向上
* メンテナンス性の向上

## プロジェクトの構成

kitsunebi は以下のような構成になっています。

- **config**
    - 設定ファイルなどを管理するディレクトリ
- **data**
    - 実データが保存されるディレクトリ
- **environment**
    - 環境変数を管理するディレクトリ
- **plugins**
    - プラグインを管理するディレクトリ
- **scripts**
    - スクリプトを管理するディレクトリ
- **secrets**
    - シークレット情報を管理するディレクトリ
- **worlds**
    - ワールドデータを管理するディレクトリ
