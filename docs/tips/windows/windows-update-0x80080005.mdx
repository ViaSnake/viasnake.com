# Windows update 時の 0x80080005 エラーについて

先日 Windows Update をしようとしたら、エラー `0x80080005` が発生してアップデートできなかった為、なんかこれしたら直ったよ的なことを書いておこうと思います。

## インターネット接続を確認

ネットに繋がってない時に、このエラーが発生するか分かりませんが、とりあえずネット接続を疑いました。例えば、ウェブサイト見れるかや速度出てるかなどなど。このページが見てる時点で基本的に接続できてると思うので大丈夫だと思いますが。

## VPNとか無効にする

VPNとか使っている場合は、無効にしてみて下さい。

## 再起動

パソコンは生き物なので再起動してあげるとよく直ります。なので、再起動してみりゃ大抵どうにかなります。

## sfcスキャンしてみる

システムがぶっ壊れてる疑惑があったので、コマンド使って修正を試みます。Windows コマンドプロンプトから下記のコマンドを実行します。

```shell
sfc /scannow
```

実行すると壊れている箇所は勝手に直してくれます。

## DISMスキャンしてみる

sfcスキャン同様にエラー直そうねってやつです。Windows コマンドプロンプトから下記のコマンドを実行してみます。

```shell
DISM /Online /Cleanup-Image /RestoreHealth
```

実行すると勝手に修復されます。
