# 利用可能な Shell 一覧を確認

Ubuntu で利用可能な Shell 一覧を確認する方法です。

```bash
cat /etc/shells
```

以下のように表示されます。

```bash
# /etc/shells: valid login shells
/bin/sh
/usr/bin/bash
/bin/rbash
/usr/bin/rbash
/bin/dash
/usr/bin/dash
/usr/bin/tmux
/usr/bin/screen
```
