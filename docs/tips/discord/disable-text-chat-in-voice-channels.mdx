# ボイスチャンネルのテキストチャットを無効化

Discord のボイスチャンネルにはテキストチャットがあります。  
この機能が邪魔（普通のテキストチャンネルを使ってほしい）と感じる事があるので、無効化する方法を説明しています。

![ボイスチャンネルのテキストチャット](https://support.discord.com/hc/article_attachments/21170474008087)

参考: [Text Channels & Text Chat In Voice Channels - Discord](https://support.discord.com/hc/en-us/articles/4412085582359-Text-Channels-Text-Chat-In-Voice-Channels#h_01HP046TF4JWJD3TWYVR5S174Y)

## 手順

1. ボイスチャンネルの `設定` を開きます。
2. `権限` を開きます。
3. `@everyone` の `メッセージを送信` を無許可にします。
