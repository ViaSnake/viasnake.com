# プロジェクトについて

プロジェクトと言っているが、実際は個人的な趣味の範囲での活動のこと。
適当な趣味よりは計画的にやろうとしていることを指す。

## このページの目的

何をやっているのか、何をやりたいのかを整理するためのページです。

その他にも、プロジェクトの進捗状況や、その他の情報を記録する。  
日々の開発や運用で必要となる手順なども記録する。

## ToDo の管理

ToDo は Linear で管理している。  
Linear と GitHub issue は連携しており、Linear で作成した issue は GitHub にも同期される。  
ただし、詳しい進捗などは Linear でしか見れない。  
あくまで GitHub issue は、どんな issue があるのかを把握するためのもの。
